# KTrek-backed-course

### Функционал
Сервис реализует логику создания курсов 

## Запуск

Для запуска сервиса необходимо установить:
- Docker
- docker-compose

### Запуск приложения
``` sh
npm starts
```

### Запуск приложения в режиме разработки
``` sh
npm dev
```

### Перезапуск приложения 
``` sh
npm restart
```

### Остановка приложения 
``` sh
npm stop
```
